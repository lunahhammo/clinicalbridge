# ClinicalBridge — Setup Guide
**For collaborators cloning this repository**

---

## Requirements

- Python 3.10 or higher
- An OpenAI API key (get one at platform.openai.com)
- ~500MB disk space for the vector store

---

## Step 1 — Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/clinicalbridge.git
cd clinicalbridge
```

---

## Step 2 — Create a virtual environment (recommended)

**Mac/Linux:**
```bash
python3 -m venv venv
source venv/bin/activate
```

**Windows:**
```bash
python -m venv venv
venv\Scripts\activate
```

---

## Step 3 — Install dependencies

```bash
pip install -r requirements.txt
```

---

## Step 4 — Add your OpenAI API key

Create a file called `.env` in the project root folder:

```
OPENAI_API_KEY=sk-your-key-here
```



---

## Step 5 — Build the vector store (run once)

This embeds the 15 patient EHR records into ChromaDB. 

```bash
python setup_vectorstore.py
```

You should see output like:
```
=== ClinicalBridge Vector Store Setup ===
Loaded 15 patient records.
Total chunks to embed: 87
Building ChromaDB at data/chroma_db/...
✓ Vector store built successfully.
```

---

## Step 6 — Run a scenario

```bash
python main.py --scenario 1
```

---

## Step 7 — Run the test suite (no API key needed)

```bash
python -m pytest tests/test_agents.py -v
```

Expected result: **47 tests passed**

---

## Step 8 — Run the evaluation harness (no API key needed)

```bash
python evaluation/test_scenarios.py --mock
```

Expected result: **5/5 scenarios passed, 40/40 checks passed**

---

## Step 9 — View the demo notebook (no API key needed)

Open `demo/ClinicalBridge_Demo.ipynb` in Jupyter or VS Code.
The notebook uses pre-captured outputs — no API calls required.

**Install Jupyter if needed:**
```bash
pip install jupyter
jupyter notebook demo/ClinicalBridge_Demo.ipynb
```

## Troubleshooting

**"OPENAI_API_KEY not found"**
→ Make sure your `.env` file is in the project root folder (same level as `main.py`)

**"Vector store not found"**
→ Run `python setup_vectorstore.py` first

**"ModuleNotFoundError"**
→ Make sure your virtual environment is activated and you ran `pip install -r requirements.txt`

**Jupyter not opening the notebook**
→ Run `pip install jupyter` then `jupyter notebook`
